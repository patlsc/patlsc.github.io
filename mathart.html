MathArt

https://github.com/patlsc/MathArt

MathArt is a series of C++ scripts that I am making for creating mathematical artwork. It is high performance and multithreaded, and will support 2d and 3d functions as well as stochastic and dynamical systems.

Differential Equations

I previously made an interactive web-based differential equation renderer https://patlsc.github.io/diffeq.html, however its performance became stretched when doing massive (QHD, 4k, etc.) renders. Whereas the javascript one takes a few seconds for most images, MathArt can render it in real time. I aim to add a GUI for the C++ side in the future.

Fractals

The mandelbrot set is the subset of the complex numbers given by

\[ \{ c \in \mathbb{C} : |f_c(0)|,|f_c^2(0)|,... \text{ is bounded} \}, \quad f_c(z)=z^2+c \]

More specifically, for rendering a mandelbrot one defines the mandelbrot function F as the number of iterations when it exceeds some massive number M

\[ F(c) = \min \{ k \in \mathbb{N} : |f_c^k(0)|>M \} \]

In fact, you can create mandelbrot variants with other functions such as

\[ f_c(z)=z \cos(z)+c \]
