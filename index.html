Unity Dialogue Scripting System

A very work in progress attempt to make a data-based dialogue and cutscene system for Unity which is simpler but equally powerful as direct scripting. Could be placed on top of any game which can be separated into 'blocks' or nodes of activity.
>start %background = bk_default narrate: "You decide to rest for a while." fade(black) if %timeOfDay == "day": %timeOfDay = "night" else: %timeOfDay = "day" narrate: "You arrive at a split road." choose("Take the left path":b, "Take the right path":c) >b narrate: "You go left." goto d >c narrate: "You go right." goto d >d narrate: "End."
